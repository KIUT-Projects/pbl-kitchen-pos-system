### Installation guide

```shell
git clone https://github.com/KIUT-Projects/pbl-kitchen-pos-system.git
```
```shell
composer install
```
```shell
npm install
```
```shell
php artisan key:generate
```
```shell
php artisan optimize:clear
```
```shell
php artisan migrate
```
```shell
php artisan storage:link
```

### Run in localhost
```shell
php artisan serve
```
```shell
npm run dev
```
